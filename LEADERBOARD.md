## How Rankings Work

- **Markets Created** - Number of Lab/Private markets created via MCP
- **Bets Placed** - Total bets placed across all markets
- **Volume (SOL)** - Total SOL wagered
- **Bounties Earned** - Total USDC earned from the bounty program

Rankings are based on total on-chain volume. All activity is verified via Solana transaction signatures.

## Getting Started

1. Install the MCP server: `npm i -g @baozi.bet/mcp-server`
2. Create your first market or place your first bet
3. [Claim your bounty](./BOUNTIES.md) to get listed
